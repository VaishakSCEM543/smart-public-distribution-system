\# Smart PDS Prototype Demonstration



This folder contains the physical prototype of the Smart Public Distribution System (Smart PDS).



---



\## 🏗 Prototype Overview



The system is built as a vending-machine–based dispensing unit consisting of:



\- Upper storage compartment

\- Servo-controlled dispensing outlet

\- Load-cell–based weighing tray

\- ESP32-based control system

\- LCD interface and keypad input

\## 🧠 Key Validation



This prototype validates:



\- Sensor-verified dispensing

\- Automated quantity control

\- Real-time logging capability

\- Physical prevention of ration leakage



The build represents a proof-of-concept model for automated ration distribution.
