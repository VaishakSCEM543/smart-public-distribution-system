The Smart Public Distribution System (Smart PDS) includes

three role-based web portals to support transparency,

accountability, and ease of operation.



\## 1. Customer Portal

\*\*Path:\*\* `web/customer/index.html`



\- Used by PDS beneficiaries

\- Displays entitlement and transaction status

\- Confirms successful ration dispensing

\- Designed for simplicity and quick access



\## 2. Shop Owner Portal

\*\*Path:\*\* `web/shop-owner/index.html`



\- Used by ration shop operators

\- Shows daily transactions

\- Displays inventory status

\- Reduces manual intervention during distribution



\## 3. Authority Portal

\*\*Path:\*\* `web/authority/index.html`



\- Used by government authorities

\- Provides real-time monitoring of ration shops

\- Shows analytics, alerts, and audit logs

\- Enables transparency and governance



All portals are connected to a common cloud backend

(Firebase) for real-time data synchronization.
