\# ESP32 Firmware – Smart PDS



Modular firmware architecture for Smart Public Distribution System.



\## Modules

\- WiFi Manager

\- Firebase Communication

\- Authentication

\- Load Cell Handling

\- Display Management



\## Platform

ESP32 using Arduino IDE



\## Setup

1\. Copy config.example.h → rename to config.h

2\. Add WiFi and Firebase credentials

3\. Select ESP32 board

4\. Upload firmware
